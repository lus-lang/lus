This page takes you from install to your first running program.

## Install automatically

On Linux and macOS, run this command in your favorite terminal emulator:

```bash
curl -fsSL https://lus.dev/install.sh | sh
```

On Windows, download and run **[lus-setup.exe](https://github.com/lus-lang/lus/releases/latest/download/lus-setup.exe)**, then follow the prompts.

On platforms without prebuilt binaries (the BSDs, illumos, Intel Macs), build from source with [Compiling Lus](/manual/compiling).

## Hello, Lus

Create a file named `hello.lus`:

```lus
global print

local name = "Lus"
print(`Hello, $name!`)
```

Run it:

```bash
lus hello.lus
```
